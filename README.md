# Robotic IV Pole

Autonomous patient-following IV pole system.

## System Components
- Jetson Nano
- Arduino Mega 2560
- SICK TiM561 LiDAR
- 12 Ultrasonic Sensors
- 4x 12V DC Motors
- Custom Power Distribution

## Repository Structure
- hardware/ → Schematics, PCB, mechanical
- firmware/ → Arduino + motor drivers
- software/ → ROS2, perception, navigation
- docs/ → Architecture, power budget
- simulations/ → Control simulations
